# Telegram Bot Commands

## Available Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot and get welcome message |
| `/help` | Show available commands |
| `/matches` | Get upcoming World Cup matches |
| `/predictions` | Get agent predictions for next match |
| `/leaderboard` | Show current agent leaderboard |
| `/agents` | Show agent status and stats |
| `/live` | Check if any matches are live now |
| `/status` | Get system status and last update |

## Example Usage

```
/start - Welcome message
/help - List all commands
/matches - Show upcoming World Cup fixtures
/predictions - Agent predictions for the next match
/leaderboard - Who's winning?
```
